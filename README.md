# SIDIntel

This is the official repository for SIDIntel, a strategic intelligence platform.

## Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/YOUR-GITHUB-USERNAME/SIDIntel.git
   cd SIDIntel
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Run the project:
   ```sh
   npm start
   ```
